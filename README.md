# Symfony E-Commerce

## Requirements

Before you begin, ensure you have the following tools installed:

- [PHP](https://www.php.net/) (Version 8.0 or higher)
- [Composer](https://getcomposer.org/)
- [MySQL](https://www.mysql.com/)
- [Symfony CLI](https://symfony.com/download)
